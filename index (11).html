📋 Pasos para activar la nube (gratis):

1. Ir a console.firebase.google.com

2. Crear proyecto → Continuar → Deshabilitar Analytics → Crear

3. En "Authentication" → Iniciar → pestaña "Sign-in method" → habilitar Anónimo

4. En "Firestore Database" → Crear base de datos → Modo prueba → Listo

5. Ir a ⚙️ Configuración del proyecto → "Tu app" → Registrar app web → Copiar config

6. Pegar la config JSON aquí abajo y guardar
