Sincronización en la Nube (Firebase)

🚨 Abrís el archivo como file:// — la nube NO puede funcionar así.

Firebase necesita una URL https://. Solución gratis en 5 min con GitHub Pages:

  1. Entrá a github.com → crear cuenta gratis
  2. Nuevo repositorio público → nombre: bertone-app
  3. Subir este archivo renombrado como index.html
  4. Settings → Pages → Branch: main → Save
  5. URL: https://tuusuario.github.io/bertone-app → compartir con todos 📱

📋 Pasos para activar la nube (gratis):

1. Ir a console.firebase.google.com

2. Crear proyecto → Continuar → Deshabilitar Analytics → Crear

3. En "Authentication" → Iniciar → pestaña "Sign-in method" → habilitar Anónimo

4. En "Firestore Database" → Crear base de datos → Modo prueba → Listo

4b. En "Storage" → Comenzar → Modo prueba → Listo (para guardar fotos)

5. En "Authentication" → Sign-in method → Anónimo → Habilitar.

⚠️ Si solo funciona en UN dispositivo:

Las reglas en "modo prueba" vencen a los 30 días. Copiá estas reglas permanentes:

Firestore → Reglas:

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read, write: if request.auth != null;
    }
  }
}

Storage → Reglas:

rules_version = '2';
service firebase.storage {
  match /b/{bucket}/o {
    match /{allPaths=**} {
      allow read, write: if request.auth != null;
    }
  }
}

5. Ir a ⚙️ Configuración del proyecto → "Tu app" → Registrar app web → Copiar config

6. Pegar la config JSON aquí abajo y guardar

💡 Para compartir con otros celulares: guardá la config y usá el botón "Compartir Config"
